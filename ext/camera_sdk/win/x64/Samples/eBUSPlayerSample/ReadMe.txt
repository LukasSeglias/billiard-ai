eBUS Player
=============

This sample is contains the source code for the eBUS Player application, used to detect, 
connect and configure GigE Vision and USB3 Vision devices.


1. Introduction

This code illustrates how to use the eBUS SDK in depth.


2. Pre-conditions

This sample assumes that:

 * You have used the eBUS Player application before
 * You have a very good understanding of C++ and Microsoft's MFC foundation classes
