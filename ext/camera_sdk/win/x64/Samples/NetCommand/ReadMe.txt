                               ==============
                                 NetCommand
                               ==============

This sample is contains the source code for the NetCommand application, used to detect, 
connect and configure multiple GEV devices.

1. Introduction

This code illustrates how to use the eBUS SDK in depth.

2. Pre-conditions

This sample assumes that:

 * You have used the NetCommand application before
 * You have a very good understanding of C++ and Microsoft's MFC foundation classes
 * You are using Visual Studio 2010 (VS10)
