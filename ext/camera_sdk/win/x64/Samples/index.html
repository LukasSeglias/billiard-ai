eBUS SDK C++ Code Samples

The following sample programs are provided to help illustrate various features of the C++ API for the eBUS SDK. Before changing the code for any of the samples, Pleora recommends that you copy the sample from the installation directory to your personal Visual Studio work space.

Sample Description
EbSetupLibSample Shows how to display information about network configuration and apply eBUS drivers to network adapters on a given PC.
eBUSPlayerSample Contains the source code for the eBUS Player application which is used to detect, connect and configure a GigE Vision or USB3 Vision devices.
NetCommand Contains the source code for the NetCommand application which is used to detect, connect and configure multiple GigE Vision and USB3 Vision devices simultaneously.
ConfigurationReader Illustrates how to use the PvConfigurationReader and PvConfigurationWriter classes.
DeviceFinder Illustrates how to use the PvSystem, PvInterface, PvDeviceInfo and PvDeviceFinderWnd classes to detect and enumerate GigE Vision and USB3 Vision devices.
GenICamParameters Shows how to use the PvGenParameter of PvGenParameterArray.
MulticastMaster Shows how to configure a GigE Vision device to stream to a multicast group.
MulticastSlave Shows how to join a multicast group and receive a stream.
MultiSource Shows how to use the PvPipeline class for image acquisition from a multi source device.
PvPipelineSample Shows how to use the PvPipeline class for image acquisition.
PlcAndGevEvents Shows how to control and handle PLC states and handle GigE Vision events.
ConnectionRecovery Illustrates how to create a simple command line application performing typical image acquisition that supports automatic recovery.
SerialBridge Shows how to use the PvSerialBridge class control a serial device connected to a Pleora Video Interface.
DeviceSerialPort Shows how to use the PvSerialPortIPEngine class for serial communication through a Pleora Video Interface.
SimpleGUIApplication Shows a typical simple eBUS SDK application.
PvStreamSample Shows how to use the PvStream class for image acquisition.
TransmitProcessedImage Illustrates how to receive video from a GigE Vision or USB3 Vision device, resample it, apply a text overlay, and retransmit.
TransmitTestPattern Illustrates how to use the PvVirtualDeviceGEV and PvTransmitterGEV classes to transmit a test pattern.
TransmitTiledImages Shows how to take four incoming receive streams and combine them into a single, tiled transmit stream.
ImageProcessing Shows how to use the eBUS SDK to acquire images in buffers managed by an external image processing library.
TapReconstruction Shows how to use the PvTapFilter class to perform tap reconstruction on PvBuffer image.
DirectShowDisplay Shows how to setup a live display from the eBUS SDK DirectShow filter.
CameraBridge Shows how to use the PvCameraBridge class to control frame grabber cameras using Pleora Camera Files, CL Protocol or GenCP.
TransmitChunkData Shows how to use PvTransmitter to transmit a mix of of images and chunk buffers.
